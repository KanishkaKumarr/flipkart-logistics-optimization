# Logistics Optimization & Delivery Performance – Flipkart

**Tools:** SQL, Excel  
**Project Type:** Data Analysis  

## Objective
Analyze delivery efficiency, warehouse performance, and shipment delays to optimize operations and reduce bottlenecks.

## Steps
1. Extracted delivery and warehouse data using SQL  
2. Calculated KPIs: average delay, on-time %, route efficiency  
3. Identified operational bottlenecks and inefficiencies  
4. Recommended optimization strategies to improve delivery performance  

## Outcome
Suggested actionable improvements with potential to reduce delivery delays by 10–15%.

## Files
- `flipkart_logistics.sql` – SQL queries for analysis  
- `kpi_analysis.xlsx` – Excel sheet with KPI calculations  
- `dashboard.png` – Visual summary of key performance metrics
